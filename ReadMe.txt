Dans ma grande bonté je vous aiderai quand meme 

-Pour lancer le front on doit faire: expo start.
-Pour lancer le back on doit faire: node App.js
-La base de donnée initiale est sur serveur/test.sql
-Pour acceder au back une fois lancé il faut faire localhost:3000/ suivi de la route demandée
-Les logiciels a installer sont: nodeJS, react native, git, visual studio, xampp.
-Pour acceder a la BDD faire localhost:8000/phpmyadmin avec xampp allumé
-Le port de xampp est 8000 par default. si le port est deja occupé vous pouvez changer la configuration du port et mettre 8080

Bonne chance.
